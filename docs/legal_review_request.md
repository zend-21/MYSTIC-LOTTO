법률 자문 요청서

모바일 앱 "미스틱 로또(Mystic Lotto)" 재화 시스템 법적 검토 의뢰

─────────────────────────────────────────────────────────────────


1. 서비스 개요

서비스명: 미스틱 로또 (Mystic Lotto)
플랫폼: 모바일 앱 (iOS / Android)
서비스 성격: 사주, 타로, 점성술 등 운세 정보 제공 및 로또 번호 추천 서비스

본 앱은 사용자의 생년월일 및 사주 정보를 기반으로 로또 번호를 추천하는 정보를 제공하는 서비스입니다. 앱 내에서 실제 복권(로또)을 구매하거나 배팅하는 기능은 존재하지 않으며, 당첨 여부와 무관하게 앱은 일절 금전적 보상을 제공하지 않습니다. 제공하는 모든 서비스는 정보 제공(콘텐츠) 서비스에 한정됩니다.


─────────────────────────────────────────────────────────────────


2. 재화 구조 (2단계 이원 화폐 시스템)

본 앱은 성격이 다른 두 종류의 앱 내 재화를 운용합니다.


▣ 나디르 (Nadir) — 유료 예치 재화

├──────────────┼─────────────────────────────────────────────────┤

▣ 재화 흐름 요약

  현금
   │ (인앱결제)
   ▼
  나디르  ← 환불 가능 (미사용분에 한함)
   │ (단방향 전환만 가능, 역전환 불가)
   ▼
  루멘   ← 환불 불가, 현금화 불가
   │
   ▼
  서비스 이용 / 봉헌 / 사용자 간 선물

루멘에서 나디르로의 역전환은 시스템상 불가능하며, 이를 약관에 명시할 예정입니다.

3. 루멘 취득 방법 상세


▣ 나디르 전환 (유상)

사용자가 나디르를 루멘으로 전환할 시 일정 비율로 루멘을 지급합니다. 전환 시점에 "디지털 콘텐츠 구매 완료 / 환불 불가"에 대한 동의 팝업 및 체크박스를 통해 별도 동의를 수령합니다.


▣ 출석 보상 (무상)

매일 앱 접속 시 소량의 루멘을 무상 지급합니다. 별도 결제 없이 누적 가능합니다.


▣ 봉헌 시스템 (확률 기반 무상 취득)

사용자가 보유 루멘의 일부를 "봉헌"에 사용하면, 랜덤 배율(예: 1배~10배)로 루멘이 반환되는 확률 기반 시스템입니다. 배율별 확률은 앱 내에 공시할 예정이며, 봉헌 결과는 앱 내 루멘으로만 귀속되고 현금화는 불가합니다.


▣ 사용자 간 선물 (무상 수령)

채팅방 등에서 다른 사용자가 루멘을 선물로 전송하는 기능입니다. 선물받은 루멘 역시 현금화는 불가하며 앱 내 사용만 가능합니다.

5. 핵심 운영 원칙 (약관 명시 예정 사항)

① 단방향 전환 원칙
나디르→루멘 전환만 가능하며, 역방향(루멘→나디르) 전환은 불가합니다.

② 현금화 불가 원칙
루멘은 어떠한 방법으로도 현금, 나디르 또는 외부 재화로 교환할 수 없습니다.

③ 전환 시 청약철회 불가 고지
나디르→루멘 전환 전, 별도 동의 화면에서 "전환 즉시 디지털 콘텐츠 구매 완료로 간주되어 전자상거래법 제17조 제2항 제5호에 따라 청약철회가 제한됩니다"라는 내용에 대한 체크박스 동의를 수령합니다.

④ 당첨 무보장 명시
서비스는 통계적·운세적 추론에 기반한 정보 제공이며, 로또 당첨을 보장하지 않습니다.

⑤ 봉헌 확률 공시
봉헌 시스템의 배율별 확률을 앱 내에 고지합니다.

⑥ 미사용 나디르 환불
루멘으로 전환되지 않은 나디르 잔액은 소비자 요청 시 환불합니다.

6. 법적 검토 요청 사항

다음 6가지 사항에 대한 법률적 견해를 요청드립니다.


▣ 검토 1. 재화 구조의 적법성

나디르(유료 예치)→루멘(앱 내 재화) 이원 구조가 대한민국 전자상거래법 및 전자금융거래법상 문제가 없는지 검토를 요청드립니다.


▣ 검토 2. 선불전자지급수단 해당 여부

나디르가 전자금융거래법상 "선불전자지급수단"에 해당하여 금융위원회 등록 의무가 발생하는지 여부, 그리고 해당 시 소규모 사업자 면제 요건 충족 가능 여부를 검토해 주시기 바랍니다.


▣ 검토 3. 봉헌 시스템의 사행성 해당 여부

루멘을 투입하여 랜덤 배율로 루멘을 반환받는 봉헌 시스템이 사행행위규제 및 처벌 특례법 또는 게임산업진흥에 관한 법률상 사행성 게임물에 해당하는지 여부를 검토해 주시기 바랍니다.


▣ 검토 4. 로또 번호 추천 서비스의 적법성

사주·통계를 기반으로 로또 번호를 추천하는 정보 제공 행위가 복권 및 복권기금법 또는 사행행위 관련 법령에 위반되는지 여부를 검토해 주시기 바랍니다.


▣ 검토 5. 카드사 차지백 발생 시 법적 대응

나디르→루멘 전환 시 충분한 고지 및 동의를 수령한 경우, 소비자가 카드사에 차지백을 신청하였을 때 해당 동의 기록이 법적으로 유효한 소명 자료로 인정받을 수 있는지 여부 및 추가 대비책을 검토해 주시기 바랍니다.


▣ 검토 6. 사용자 간 루멘 선물의 법적 성격

사용자 간 루멘 선물 기능이 전자금융거래법상 "자금이체"에 해당하여 별도 규제를 받는지 여부를 검토해 주시기 바랍니다.

7. 참고 사항

- 유사 서비스 사례: 카카오페이지 캐시/쿠키, 네이버웹툰 쿠키, 넥슨 캐시/NX포인트 등 이원 재화 구조 운용 중
- 앱 내 실제 복권(로또) 구매 기능 없음
- 앱 내 사용자 간 현금 거래 기능 없음
- 루멘의 현금화(출금) 기능 없음
